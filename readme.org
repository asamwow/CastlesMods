** Bootstrap

Use this code to setup your mods folder to track changes.

#+begin_src sh :tangle bootstrap.sh
git init &&
git remote add origin https://github.com/asamwow/CastlesMods.git &&
git fetch &&
git reset --soft origin/main &&
git add . &&
git status
#+end_src
